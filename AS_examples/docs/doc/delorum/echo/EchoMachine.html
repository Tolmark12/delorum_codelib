Packagedelorum.echo
Classpublic class EchoMachine

This class is used to report errors or other messages independent of how the swf has been deployed. For instance, if the swf has been embedded in an html page, the errors and messages are sent as Javascript alerts; When deployed to the flash runtime, as flash trace statements.


Example 
 
 // Determine what mode to output errors and messages
 ErrorMachine.setechoModeAutomatically( this.stage );
 // or, to set the error mode manually... 
 // ErrorMachine.echoMode = ErrorMachine.FLASH;
 
 // Add a sample error
 ErrorMachine.addErrorToLog( "This is a fake error." );
 // Print all errors
 ErrorMachine.printErrors();
 // Outputs:
 // >> Errors
 // This is a fake error.
 // << End
