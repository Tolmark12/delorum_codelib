reflect()method 
public function reflect($surfaceShinyness:Number = 0.2, $surfaceReflectivity:Number = 0, $reflectionHeight:Number = 1, $surfaceBlur:Number = 0, $verticalShift:Number = 0):void

Create an image reflection

Parameters
$surfaceShinyness:Number (default = 0.2) — 0-1 : How "shiny" the surface is. ie: how faded the reflection is
 
$surfaceReflectivity:Number (default = 0) — 0-1 : How "reflective" the surface is. ie: how much gradient. A value of 1 would show the gradient sitting on a mirror.
 
$reflectionHeight:Number (default = 1) — 0-1 : Percentage of the original image's height. 1 would have a reflection as tall as the image. 0.2 would reflect the bottom 20% of the image
 
$surfaceBlur:Number (default = 0) — Amount of blur
 
$verticalShift:Number (default = 0) — Vertical Shift - The distance to move the reflection up
